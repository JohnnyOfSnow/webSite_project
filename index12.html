環境安裝與第1個程式

歡迎來到程式的世界，在本頁中將要教你要如何寫基本的java程式
所謂工欲善其事，必先利其器，要寫java程式之前，必須安裝一些東西，設定開發程式的環境，接下來，我們從設定開發程式的環境開始。

ㄧ、環境安裝

首先，你可能在使用瀏覽器時有遇到java的安裝，但那個並不是我們這裡講的開發java的環境，因為有些網頁上的程式需要用到java一些元件，所以這個僅能讓我們觀看。
現在，請按照下面的步驟，完成開發java的環境的安裝。

首先，請打開瀏覽器收尋jdk

jdk代表java develop kit，這個裡面有開發java程式所需的元件

然後請點選這一個

進去之後，請在那個頁面找到這裡

請先點選OAccept License Agreement，然後你才可以點選下面下載，根據你的電腦作業系統以及位元數選擇。

下載完後，請對著剛剛下載的檔案點2下完成安裝。

到此，我們已經完成安裝所需的開發環境，接著，我們要設定環境

請先找到你剛剛安裝的jdk路徑，然後打開bin資料夾，將現在的位置複製起來。

例如：C:\Program Files\Java\jdk1.7.0_45\bin

然後找到我的電腦

將所有視窗都按下確定之後，到此已經完成設定環境的步驟。

接著，我們開始寫第1個程式，並且執行它。

二、第1個程式

寫程式的步驟：1.想你要寫什麼，2.開啟文字編輯程式(寫程式) 3.編譯  4.執行

1.想你要寫什麼

這裡我們是第1次寫程式，所以我們先練習印出"第1個程式"這5個字。

2.開啟文字編輯程式(寫程式)

請打開這個，或是任何可以輸入文字的軟體都可以

在裡面開始輸入下面的內容

public class Test{
	public static void main(String[] args){
		System.out.println("第1個程式");
	}
}

完成之後，請存檔。

程式說明：

public class 檔名{
	public static void main(String[] args){
	}
}

上面的部份，是每ㄧ個java程式所需要的部份，其中public class 之後的字，決定存檔的檔名，在public static void main(String[] args){}的左右括弧內我們可以想要做的事，剛剛打的System.out.println("第1個程式");，其中System.out.println()這個是印出被""包的文字的敘述，這個程式執行完後，預期會印出第1個程式這5個字。

到此我們已經完成編寫程式的動作，接著要執行程式。

首先，請打開這個，找不到的話，可以用搜尋cmd的方式。

再來將你剛剛寫的程式找到它的位置並且複製

例如：C:\Documents and Settings\Genuine\桌面

在命令提示字元視窗中輸入

cd 你剛剛複製的位置

然後按下enter

有沒有發現字不一樣，現在這個動作是將編譯位置移動到程式的位置

什麼叫做編譯呢?簡單來說，是要讓電腦看你的程式之後，改成電腦看的懂的字，以方便電腦執行。

3.編譯

接著我們要進行編譯的動作

在命令提示字元視窗中繼續輸入

javac Test.java

然後按下enter

剛剛按下enter後，你發現到有過一些時間後才跳出下一行，在這段時間中電腦正在進行編譯的動作，另外ㄧ件事，你發現到和程式同位置的地方多了一個檔案，這就是剛剛說的(電腦看你的程式之後，改成電腦看的懂的字，以方便電腦執行。)，它做了類似一種筆記的方式，上面寫著程式要如何執行(但對我們人，我們不是看的懂)。

4.執行

編譯完了之後，我們可以執行程式了。

在命令提示字元視窗中繼續輸入

java Test

然後按下enter

我們看到剛剛在程式中的"第1個程式"這5個字，出現在視窗上了。

到此我們完成執行程式了。
