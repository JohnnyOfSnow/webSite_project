五、選擇式介紹

在程式中執行時會按照敘述的先後順序一一執行，這種我們稱為循序
但有時候，我們想要在中間控制，改變它的執行順序，有些敘述不要做的時候
選擇的概念就出來了，藉由判斷條件的方式控制執行順序。
這裡，我們將會介紹3種選擇式。
但在介紹之前，我們先介紹一些如何表達的敘述。
生活中的表述程式中的表達
a大於3a > 3
a小於3a < 3
a大於等於3a >= 3
a小於等於3a <= 3
a等於3a == 3
a不等於3a != 3

以上的表告訴我們該如何在程式中表示生活中的比較大小相等不相等，值得注意的是常常a等於3在判斷式時很多人會寫成a = 3(這是將3這個值給a)，若要表達相等的意思為2個"="。

ㄧ、if

提供了判斷ㄧ種條件的選擇敘述

語法:
if(判斷式){
裡面寫符合判斷式時要執行的敘述
}

例如：
if(bd == 4){
bd = bd + 2;
}

在上面的例子中，我們寫了一個選擇式，當執行到if(bd == 4){時，會判斷bd是不是等於4呢?如果是就進入if{}裡，如果不是就直接到}，然後離開if

例如：
if(bd == 4){
bd = bd + 2;
}else{
裡面寫不符合判斷式時要執行的敘述
}

這個例子，將上述的多加了else敘述，當bd是不是不等於4，就會執行else{}裡的敘述。

以下是一個模擬if執行的操作，你需要在(請輸入一個整數值給 i)右邊的格子內輸入一個整數，接著，你可以按下ㄧ步來看如何執行，按下全部清除將會全部清空，你就可以再使用ㄧ次。

int i;

if(i == 3){

如果 i 等於 3 就執行這裡

}

if 的外面
請輸入一個整數值給 i

二、if-elseif-..-else

提供了選擇多種條件的敘述

在上面的if選擇敘述，只能以一種條件判斷，如果我們需要判斷多種條件時該怎麼辦呢?，這裡的if-elseif-..-else將可以讓你設定多種條件。

語法:
if(判斷式1){
裡面寫符合判斷式1時要執行的敘述
}else if(判斷式2){
裡面寫符合判斷式2時要執行的敘述
}else if(判斷式3){
裡面寫符合判斷式3時要執行的敘述
}else{
裡面寫都不符合前面的判斷式時要執行的敘述
}

例如：
if(bd == 4){
bd = bd + 2;
}else if(bd == 6){
bd = bd + 4;
}else if(bd == 8){
bd = bd + 8;
}else if(bd == 10){
bd = bd + 12;
}else{
bd = bd + 17;
}

在上面的例子中，我們寫了一個選擇式，當執行到if(bd == 4){時，會判斷bd是不是等於4呢?如果是就進入if{}裡，如果不是就到}else if(bd == 6){判斷bd是不是等於6呢?，如果是就執行下一行，如果不是就到}else if(bd == 8){，若bd都不滿足所有條件，就執行else{}裡的敘述

if-elseif-..-else的這種選擇式中間的elseif()可以寫很多個，但是最後一定要有else，否則程式會出錯。

以下是一個模擬if-elseif-else執行的操作，你需要在(請輸入一個整數值給 a)右邊的格子內輸入一個整數，接著，你可以按下ㄧ步來看如何執行，按下全部清除將會全部清空，你就可以再使用ㄧ次。

int a;

if(a == 4){

a = a + 2;

}else if(a == 6){

a = a + 6;

}else{

a = a + 3;

}

if-else的外面
請輸入一個整數值給 a

三、switch-case

提供了選擇多種條件的敘述，比起if-elseif-..-else更容易看。

語法:
switch(要判斷的變數){
case 值1:
裡面寫符合值1時要執行的敘述
break;
case 值2:
裡面寫符合值2時要執行的敘述
break;
default:
裡面寫都不符合前面時時要執行的敘述
}

例如：
switch(bd){
case 1:
bd = bd + 2;
break;
case 2:
bd = bd + 8;
break;
default:
bd = bd + 17;
}

在上面的例子中，我們寫了一個選擇式，當執行到switch(bd){，會以下的判斷以bd的值來看，case 1:指的是bd是不是等於1，如果是就執行下一行，如果不是就到case 2:，繼續做bd是不是等於2，若bd都不滿足所有條件，就執行default裡的敘述，值得注意的是break;，當程式執行到此，會直接跳到最底(})，而離開選擇式。

看了switch後，有沒有發現它比if-elseif-else還來的容易看，這個我們稱為程式具有可讀性。

以下是一個模擬switch執行的操作，你需要在(請輸入一個整數值給 k)右邊的格子內輸入一個整數，接著，你可以按下ㄧ步來看如何執行，按下全部清除將會全部清空，你就可以再使用ㄧ次。

int k;

switch(k){

case 2:

k = k + 4;

break;

case 4:

k = k + 6;

break;

case 6:

k = k + 8;

break;

default:

k = k + 10;

}

switch的外面
請輸入一個整數值給 k
