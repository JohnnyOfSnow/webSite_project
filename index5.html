五、迴圈(loop)介紹

你現在已經學會如何在程式中表達判斷的敘述，現在在這裡要學到重複執行的敘述
迴圈，是在程式中表達重複的敘述。以下將要介紹3種迴圈

1.for loop

2.while loop

3.do-while loop

ㄧ、for loop

語法:
for(初始式;判斷式;算式){
在for loop裡面寫你想重複做的事

}
例如：
for(int count = 0; count <= 14;count = count + 1){
}

在上面的例子中我們定義了一個for loop，(初始式)從0開始，每執行完1次for loop(就是從for loop的左大括弧到右大括弧之間)，將count + 1(算式)，這時會先做判斷count是不是小於等於14呢?如果是，就再做for loop的左大括弧到右大括弧的範圍內的敘述，直到，最後判斷時不符合count是不是小於等於14，才離開for loop

以下是一個模擬for loop執行的操作，你需要在(請輸入一個整數值給 o)右邊的格子內輸入一個整數，接著，你可以按下ㄧ步來看如何執行，或是按下ㄧ次做完，按下全部清除將會全部清空，你就可以再使用ㄧ次。

int o;

for(int i = 0; i <= 10; i++){

o = o + 2;

}

for 迴圈的外面
請輸入一個整數值給 o

二、while loop

語法:
while(判斷式){
在while loop裡面寫你想重複做的事

}
例如：
int count = 0;//(初始式)
while(count <= 14){//(判斷式)
count = count + 2;//(算式)
}

在上面的例子中我們定義了一個while loop，(初始式)從0開始，一開始進入while loop，先判斷count是不是小於等於14呢?如果是，執行while loop左大括弧到右大括弧裡的敘述，當執行到右大括弧時，再判斷count是不是小於等於14呢?，如果是，就再執行while loop內部，直到count不符合(是不是小於等於14)，就離開while loop

以下是一個模擬while loop執行的操作，你需要在(請輸入一個整數值給 w)右邊的格子內輸入一個整數，接著，你可以按下ㄧ步來看如何執行，或是按下ㄧ次做完，按下全部清除將會全部清空，你就可以再使用ㄧ次。

int w;

int count = 1;

while(count <= 14){

w = w + 5;

count = count + 1;

}

while 迴圈的外面
請輸入一個整數值給 w

三、do-while loop

語法:
do{
在do while loop裡面寫你想重複做的事

}while(判斷式);
例如：
int count = 0;//(初始式)
do{
count = count + 2;//(算式)
}while(count <= 30);//(判斷式)

在上面的例子中我們定義了一個do while loop，(初始式)從0開始

第1次就直接進入do-while loop

遇到迴圈最底，判斷count是不是小於等於30，如果是，再執行do while loop內部，直到count不符合(count是不是小於等於30)，就離開do while loop

以下是一個模擬do-while loop執行的操作，你需要在(請輸入一個整數值給 d)右邊的格子內輸入一個整數，接著，你可以按下ㄧ步來看如何執行，或是按下ㄧ次做完，按下全部清除將會全部清空，你就可以再使用ㄧ次。

int d;

int count = 1;

do{

d = d - 2;

count = count + 4;

}while(count <= 30)

while 迴圈的外面
請輸入一個整數值給 d
