二、定義類別、變數以及方法

ㄧ、類別

對物體的ㄧ個概說
定義類別，是因為找到那種物件擁有的靜態特徵，與動態行為
在生活中我們是不是能分別各種事物？
憑藉的是他們有各自的特徵(Attributes)和行為(behavior)
利用以上說的2點我們可以描繪出這個事物的樣子
這個概念我們稱為類別化
經由上述的過程，我們可以撰寫類別。

定義的語法如下

class 類別名字{

}

在上述舉例中，定義一個類別需要使用"class" 這個字，後面加上名字 接著寫上{}，在大括弧範圍內寫入變數(靜態特徵)，方法(動態行為)。

二、定義變數

物件的特徵我們定義成變數
變數有下面的定義規則

變數的資料型態 變數名子 = 值;

	變數的資料型態有：	變數名子有以下的規定：
		整數  int		1.不可以開頭為數字
		小數  double		2.不可以用java已經定義的字(reserved word)
		浮點數  float		3.可以用 $  _ 當開頭
		字元 char					
		整數陣列 int[]
		小數陣列 double[]
		字元陣列 char[]

定義變數的名字我們希望能從名字望文生義

而使程式增加可讀性。

例如 int radius = 3;

在上述的例子中，我們定義了一個變數叫做radius，它的資料型態是整數(int) 而這個變數它的值是3。
